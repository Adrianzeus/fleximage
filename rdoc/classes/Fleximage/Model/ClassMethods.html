Module Fleximage::Model::ClassMethods
In: lib/fleximage/model.rb

Provides class methods for Fleximage for use in model classes. The only class method is acts_as_fleximage which integrates Fleximage functionality into a model class.

Methods

acts_as_fleximage  

Public Instance methods

acts_as_fleximage(options = {})

Use this method to include Fleximage functionality in your model. It takes an options hash with a single required key, :image_directory. This key should point to the directory you want your images stored on your server.
